# AI Heroes Website

The marketing website for [AI Heroes](https://aiheroes.nl) — an AI consulting company helping organizations navigate AI with practical workshops and strategic insight.

## Tech Stack

- **React 19** with TypeScript
- **Vite** for development and builds
- **React Router** for client-side routing
- **Tailwind CSS** for styling

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
├── components/     # Reusable UI components
├── pages/          # Page components
│   ├── en/         # English pages
│   └── nl/         # Dutch pages
├── constants.ts    # Content and translations
├── types.ts        # TypeScript definitions
└── App.tsx         # Router configuration
```

## Deployment

The site is built as a static single-page application and can be deployed to any static hosting provider.
